# Task 2 - C++ Programming Assignment

## Description
This project contains solutions to basic C++ programming tasks using:
- Arrays
- Pointers
- References
- Structures

## Files

### 1. array.cpp
- Demonstrates array creation
- Uses:
  - Normal for loop
  - For-each loop

### 2. pointer.cpp
- Demonstrates pointer usage:
  - Address storage
  - Value access
  - Value modification

### 3. reference.cpp
- Demonstrates reference variables
- Modifies original variable using reference

### 4. structure.cpp
- Defines a Rectangle structure
- Calculates and prints area
